Ce projet de ma formation visé à nous apprendre l'intégration en HTML / CSS en s'appuyant sur une maquette prédéfinie. J'ai donc choisis de refaire côté FRONT-END la célèbre application INSTAGRAM. Par la suite, j'ai décidé d'utiliser une base de données real time de Firebase pour y stocker des données.

    Compétences:
  • Intégration en HTML/CSS à partir d'une maquette prédéfinis.
  • Utilisation du Framework open-source IONIC
  • Utilisation de Firebase pour stocker des données avec une base de données realtime noSQL.

Création d'un bot discord en utilisant Discord.js dans le but de sauvegarder les wallets du projet Infamous Billionaire dans une base de données NoSQL MongoDB. Ces données seront par la suite réutilisées pour créer une whitelist pour la collection et seront affichées dans un google sheet afin d'assurer à la personne que son wallet a bien été sauvegardé.

    Compétences:
  • Utilisation de la bibliothèque discord.js
  • Intéragir avec une base de données noSQL (MongoDB)
  • Utilisation de l'API de Google pour manipuler un googlesheet et y inserant des données d'une base de données.
